关于羽毛球的一些真·人生经验

Posted on 2018-09-28 • In 追球

技术篇

  1. 找个靠谱的教练，认真系统的学习一段时间，是业余真正的捷径；
  2. 多打单打，会比双打更容易提高个人能力；
  3. 打好基础，知道自己打出去的球会到什么位置，才能讲预判下一拍和战术；
  4. 刻意练习很重要，每一个技术动作都需要花时间、花心思、花精力去“练球”，才能掌握的好；
  5. 基础技术动作 —> 身体素质 —> 球路 —> 战术，升级道路大概是这样的；
  6. 良好的身体素质，是速度、力量、少失误的保障，所以需要你额外做很多无球的专项训练；

伤病篇

  1. 运动前动态拉伸（热身），运动后放松（静态拉伸），是减少意外伤病最好的办法；
  2. 业余80%的膝盖伤病，都来自于膝盖力量不足，靠墙静蹲是一个锻炼的好动作；
  3. 运动前，请认真评估自己的状态，疲劳、醉酒后打球，大概是拿生命在运动吧；
  4. 受伤后，请认真养伤，莫贪一时之乐；
  5. 适量、适度运动；

装备篇：

  1. 装备不一定要买最好的，“合适”才重要；
  2. 为了膝盖和脚，鞋子要买好的，尺码和宽楦要合适；
  3. 合适的毛巾袜和鞋垫占脚感体验很重要的一部分；
  4. 觉得球拍不好打？换根线、换个磅数试试；
  5. 压缩护腿对于大跑动量的人，确实有好处；
  6. 汗多手胶滑，试试护腕；

DNS 皮毛

Posted on 2018-09-18 • In Compoter Science

这是一篇介绍 DNS（Domain Name System ） 皮毛的文章。

一、什么是 DNS

DNS = Domain Name System

我们平时访问网页，通常都是输入一个网站的域名（eg: https://google.com ），然后游览器会返回相应的网页给我们。

游览器拿到我们输入的域名，然后通过网络，最终找到相应的服务器拿到了我们要的网页，再返回给游览器。

在万维网中，服务器地址通常是用 IP 进行表示的。那么，我们的输入的域名，是怎么变成服务器的 IP 地址呢？

这个转换的过程，就是DNS 要做的。

flv.js框架解读

Posted on 2018-09-11 • In Compoter Science

知识储备

流媒体方面的知识

  • flv格式
  • mp4、fmp4格式

前端方面的知识

  • H5的video、audio标签
  • ES6
  • MSE

核心实现

  • video标签
  • MSE

FLV（Flash Video）是 Adobe 推出的一种视频格式，既可以用flash 播放，又可以用其他非flash 播放器播放。

video标签是 H5的一个新媒体标签，即视频播放标签。但是video标签不支持直接播放flv 的视频。

MSE（Media Source Extensions），即流媒体扩展 API。 提供了实现无插件且基于 Web 的流媒体的功能。使用 MSE，媒体串流能够通过 JavaScript 创建，并且能通过使用 audio 和 video 元素进行播放。

所以flv.js核心做的事情是：将获取到的flv 流媒体，经过格式转换，然后通过MSE 接口，喂给video标签。

程序员装逼(▼へ▼メ)指南

Posted on 2018-08-30 • In Compoter Science

其实程序员，是实际应用中和科研结合特别紧密的一个行业。

所以，程序员的外(zhuang)化(bi)工程就和知识管理(PKM)模型特别像。

PKM = Personal Knowledge Management

PKM 有三个步骤：

  1. 知识的获取；
  2. 知识的内建；
  3. 知识的输出；

专业知识和软实力知识的学习，是后期内建和输出的基础。学习的过程包括知识、感受的获取，然后理解，初步建立记忆体系。

内建，在初期的理解消化之后，多数是在实践中的进行应用、提炼总结获取的高价值知识、观念。

知识的输出形式则是多样的，日常工作技能的输出、写书写博当网红、国内新技术的布道…只要你前两步做的足够强大，输出的形式只有你想不到的。

这里以一个毕业工作2-3年，成长速度普通的小码农作为标本，探讨一些有效可行的外(zhuang)化(bi)手法。
