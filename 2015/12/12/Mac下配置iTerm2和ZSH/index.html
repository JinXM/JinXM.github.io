Mac下配置iTerm2和ZSH

发表于 2015-12-12   |   分类于 环境配置   |  

作为一只程序猿 , 在 Mac 或者 Linux 的环境下编程, 少不了和命令行接触. 而一个调教的好的命令行工具可以大大提升开发效率, 愉悦心情.

首先要区分一个概念, shell 和 终端模拟器.

  • shell 是指 用户和系统内核接触的一个桥梁(命令), 通常说的 shell 脚本什么就是指这个,主要的 shell 有bash 和 zsh.
  • 终端模拟器 是指一个操作终端的客户端, Mac 和 Linux 自带的 terminal 就是一个终端模拟器.

-正文

系统环境：MAC OS X
终端模拟器: iTerm 2
shell: zsh

终端模拟器

这个推荐 iTerm 2 ,虽然说 Mac 自带的 iterminal 已经很优秀了 , 但是有更优秀颜值还高的怎么会不喜欢呢.

快捷键

拿到一个新工具的第一件事, 肯定是去 了解快捷键.
这里罗列几个高频快捷键,通过快捷键可以快速了解 iTerm.

快捷键 功能
cmd + d 水平分屏成两个
cmd + shift + d 垂直分屏成两个
cmd + n 新建一个窗口
cmd + f 查找
ctrl + u 清空当前行
cmd + r 相当于 clear的效果,不过不是清楚屏幕内容而是上移内容

shell

这个严重推荐 zsh, 理由如下:

  • 兼容 bash
  • 有很多实用的优化小功能
  • 最重要的是有许多插件,可定制性很强

安装

配合 iTerm 实用的话, 推荐 Oh My Zsh <-安装/文档戳这里

主题

安装完 Oh My Zsh 后, 可以选择自己心仪的主题 .
PS: 部分主题依赖于Powerline Vim plugin,可能需要安装相应地 字体 ,并 配置iTerm 2 -> 设置 -> Profile -> Text -> Regular Font 和 Non-ASCII Font 两个的字体为 powerline 字体才能显示箭头之类的. 逼着选用的字体是 Inconsolata for Powerline

主题方面笔者选择的是 Bullet train.

插件

Oh My Zsh 有很多实用的插件可以选择. 在 Oh My Zsh 的GitHub 工程 plugins 目录含有大量插件,使用方法戳->官网文档

如果想要效(Bi)率(Ge) 更高,去学习 shell 的命令行吧,可以去看别人总结的一些实用命令行, 鸟哥的 Linux 私房菜-基础篇 是一个很不错的选择呢 !
