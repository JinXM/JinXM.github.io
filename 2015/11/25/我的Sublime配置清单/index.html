我的Sublime配置清单

发表于 2015-11-25   |   分类于 环境配置   |  

在新公司实习，拿到新电脑当然是先个性化配置一番了。在这里记录并分析一下自己的 Sublime 配置，这里只放一些在个人开发中觉得很实用的东西。

水平有限，恳请指正。

-正文

系统环境：MAC OS X
软件版本：Sublime Text3
主要用途：前端开发编辑器

1.惯例：PackageControl

这个看官网就可以了。安装完这个，就可以用它去安装其他插件了！

2.主题安装

作为一个视觉党,第一件事当然是安装一个自己中意的主题来摆脱 Sublime 枯燥的黑白色啦.

这方面，笔者选择的是itg.flat.

3.Colorcoder

一个高亮所有变量的插件.安装依旧是使用 PackageControl 进行.

4.SublimeLinter

SublimeLinter ,

这是一个规范检测的插件,笔者主要用这个插件检测 ES6 代码是否遵循 ESlint 规范.安装完插件之后,就是配置检测为 ESLint 了.

5.Markdown Editing

因为笔者时常使用 markdown ,所以一款高亮 markdown 的插件当然是必不可少的.这里推荐Markdown Editing

6.其他

上述的是笔者必备的一些插件.当然,还有一些比较出名的插件大家可以根据自己的情况选择: Emmet , Git , Terminal , Sublime CodeIntel 和 一些语言的语法高亮插件.

7.快捷键

要提高开发(Bi)效(Ge)率,除了一些优秀的插件外,敲得一手好快捷键也是重要.以下罗列以下几个使用频率比较高的快捷键(Mac):

快捷键 功能
shift + cmd +p 打开命令面板
contrl + ` 打开控制台
cmd + n 新建标签
cmd + 数字 切换标签
cmd + option +2 水平分屏成两个
control + 数字 分屏后移动到不同屏幕
cmd + delete 删除光标所在行光标前内容
cmd + t 文件跳转
control + g 跳转到指定某一行
cmd + r 函数跳转
cmd + / 给选中行 添加/去除 注释
cmd + k +b 关闭侧边栏
cmd + p 查找并打开文件
