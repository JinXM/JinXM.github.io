2018年的学车故事线终于结束啦！

2018-08-21

2017年12月19日 ~ 2018年08月20日，随着昨天科目四的 pass，我那历时整整8个月的漫长的学车故事线终于结束了！

报的魔都本地的驾校，小伙伴推荐的师傅。就这样，我进入了马路杀手培训班，开启了我漫长的学车故事线。

期间各种体会，以下是我的碎碎念：

/ 尽量在读书的时候去 get driving license ，工作后就要做好持久战的准备。

/ 你会吸很多二手烟，然后发现好烟的二手烟会比差的舒服点，特别是对于我这种有点晕烟的人来说。

/ 老教练有时候固然教历长，但是很多时候教学老旧思想保守，沟通也容易有代沟。

/ 一车人来自社会的各个角落，你会听到很多低俗笑话，说话多的人往往不是特别好的。

/ 年龄差、经历和教育程度的不同，会让你或轻或重的体会到代沟。

/ 驾驶证考试多是靠记忆和熟练度，先弄懂要点再去熟练操作会变得高效。

/ 学车时间太短容易掌握不了实际技能，太长则占用其他时间太多效率太低，所以找学车师傅前先问清楚大概时间做好评估。

/ 以上皆是瞎说。
