JS对象：函数-读书笔记

Function 概述

函数Function 是 JS 预设好的一种对象类型，函数名即指向这个对象的一个指针，所以可以把函数名当做其它函数的参数。

JS的函数是没有重载的。

函数对象默认定义好、主要的属性和方法有如下：

  • arguments 和 callee
  • this
  • call()、apply() 和 bind()
  • prototype
