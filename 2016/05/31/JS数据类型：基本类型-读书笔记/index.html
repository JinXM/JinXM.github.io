JS数据类型：基本类型-读书笔记

Posted on 2016-05-31 • In JavaScript Note

JS 数据类型

在 ES 5标准中，JS 有6个基础的数据类型。



数据类型的判断方法

可以使用 typeof操作符和 instanceof运算符来检测数据类。

typeof 主要用于检测基本的数据类型。

instanceof 主要用于检测是什么类型的 Object。

undefined 和 null

  • undefined：变量声明了但未初始化时候的值。
  • null：空指针。使用尚未声明的变量会得到 null。

Number

  1. Number 类型用来表示整数、浮点数、不同进制的数。
  2. 当超过了 Number 的数值大小限制时，用特殊值 NaN (Not a Number)表示。
  3. 有 Number()、parseInt()、parseFloat()三个函数将其他类型转换为 Number 类型。

String

  • 特殊符号转义
  • 字符串拼接：+
  • 转换为字符串：toString(value)、String()

Boolean

只有 true/false 两个值。
Boolean(value)方法可以将任意值转换为布尔值，会被转换为 false 的值有空字符串、0、NaN、null、undefined。
