ECMAScript 语言构成

  1. 关键字：var、new、if等，这些关键字在具有特定用途。
  2. 保留字：一些目前没有用途但被保留下来以后可能会被使用的词。
  3. 操作符：+、-、.等。
  4. 语句：while、for、if…else 等语句。
  5. 语法：如何使用 JS 的规则。
  6. 数据类型：定义了 JS 的数据结构。
