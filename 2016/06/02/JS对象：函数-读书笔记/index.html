JS对象：函数-读书笔记

Posted on 2016-06-02 • In JavaScript Note

Function 概述

函数Function 是 JS 预设好的一种对象类型，函数名即指向这个对象的一个指针，所以可以把函数名当做其它函数的参数。

JS的函数是没有重载的。

函数对象默认定义好、主要的属性和方法有如下：

  • arguments 和 callee
  • this
  • call()、apply() 和 bind()
  • prototype

函数没有重载

所谓重载，即已经定义了一个函数，又定义了一遍这个函数，两次定义时函数的名称不变，但是参数有所改变。这样使用时，可以根据参数的不同执行不同决定执行第一个定义还是第二个。

没有重载的理由：如前所述，函数名是一个指向对象的指针，函数参数是对象内部的内容，存储在 arguments 属性中。但定义两次，函数名指针会先指向第一个定义的对象，然后指针又重新指向后一个定义的函数对象，最终指向最后定义的那个函数对象。

不需要重载的理由：JS 函数定义时，参数会被存储在 arguments 属性中，这个 arguments 属性可以会在实际使用时动态接受参数，即函数在实际使用时的参数个数可以定义时的参数个数不一致。这就让函数运行时实际接受参数并不依赖于声明时的，所以不需要重载。

arguments 和 callee 属性

arguments 是一个类数组对象，内部存储着函数接受的对象，可以像数组一样 arguments[index]的读取参数。但 arguments 还包含一个 callee属性，这个属性是一个指向函数本身的指针，是和函数名指针一样指向同一块内存的指针。

this 属性

this是指向调用这个函数的对象(其实就是一个函数执行的环境对象)，如果是全局环境，则调用对象是 window。但是在构造函数里，this 指向的是函数的实例。

call()、apply() 和 bind()方法

这三者主要的功能是改变函数的 this 指向，即设置定义好的函数在什么环境中执行。

call()和 apply()功能完全一样，主要是改变函数的 this,仅仅第二个参数不同。bind()除了改变 this,还会创建函数的实例。
