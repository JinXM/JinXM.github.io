JS对象：基本包装类型-读书笔记

2016-06-03

基本包装类型概述

ECMAScript 5标准中，JS 有6种数据类型，Number、Boolean、String、undefined、null和 Object。Object是引用类型，其余的是基本类型，不具备对象特征(null 是空指针，可以看做 Object)。

但是在实际在读写 Number、String、Boolean 类型时，系统会动态为这三种类型包装上一层对象类型的壳，可以让我们像对象一样操作它们，在读写完后又会卸下这层壳（PS：JS中所有变量都是对象这种说法是不正确的）。这个加壳的过程如下：

  1. 读写到 Number、String、Boolean这三个基本类型变量值时，系统自动为其创建一个相应的包装对象(就是一个包含特定方法属性的对象)，包装上去；
  2. 使用包装对象提供的方法和属性进行操作，让人感觉是在使用对象一样；
  3. 当读写语句执行完，系统自动卸载掉、销毁包装对象，恢复成基本类型。

包装对象(即”壳”)首先是一个对象，包含所有对象都包含的 construcotr、hasOwnProperty()、isPrototypeOf()、prototypeIsEnumerable()、toLocalString()、toString()、valueOf() 七个方法基本方法。

此外，每个类型的包装对象还提供了该类型专门的属性和方法。

Number 包装类型

Number 的包装对象提供了的属性和方法主要是处理数字格式的，适用于货币处理这些场景：

  1. toFixed(): 小数点后保留几位小数。
  2. toExponential: 格式化为指数表示法。
  3. toPrecision： 格式化为指定位数的格式。

Boolean 包装类型

Boolean 的包装对象主要提供的对象和方法只有 Boolean()一个方法，这个方法将其他类型转换为 Boolean 值。

String 包装类型

String 的包装对象提供了的属性和方法主要是用于处理字符串格式的，包含字符查找、字符串合并、截取、比较大小、大小写转换、模式匹配和 trim。



字符查找

字符查找包括知道位置查找字符、知道字符(串)查找位置两种。

知道位置查找字符主要是 charAt()、charCodeAt()两个方法，前者返回字符，后者返回字符的编码。

知道字符(串)查找位置主要是 indexOf()、lastIndexOf()两个方法，输入需要查找的字符(串)，返回字符(串)在变量字符串中首个位置 index，前者是正向查找，后者是反向查找。这两个方法也可以用于数组中。

字符串合并

concat()方法可以用来合并字符串，也可以用来合并数组。但是在实际使用中，更多的是使用”+”运算符来拼接字符串。

字符串截取

slice()、substring()、substr()三个方法都提供了从原始字符串中截取一段字符串的功能。slice()同样还可以截取数组。

  • slice(arg0,arg1): arg0表示开始截取位置的下标，arg1表示结束位置的后一个位置。(arg1-arg0)表示截取长度。
    • substring(arg0,arg1): 参数含义同 slice()。
    • substr(arg0,arg1): arg0表示开始截取的位置，arg1表示截取的长度。

比较大小

String包装对象的localeCompare()方法可以比较两个字符串之间的大小。比较方法是从两个字符串的第一个字符开始比较，若是第一个不同则第一个编码比较大的大，若相同则比较后一个，以此类推。当前边部分相同，字符串比较长的为大者。

大小写转换

String 包装对象的大小写转换方法是 toUpperCase()、toLowerCase()；但是针对不同语言环境，还有一对 toLocaleUpperCase()、toLocaleLowerCase()方法。这和 toString()和 toLocalString()的道理一样。

匹配模式

匹配模式是使用正则来解析字符串。详见正则篇读书笔记。

trim()方法
trim()方法是将字符串头部和尾部的空格去除，然后返回一个去除空格后的字符串。
