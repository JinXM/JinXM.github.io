匿名函数

匿名函数介绍

如上所示，JS函数的声明有两种形式，直接定义和将函数赋值给一个变量。在第二种中，function(){}这种函数称之为匿名函数。之前我们提到过函数名就是一个指向函数内存区域的指针，所以单独声明 function(){}这种形式是没有指向这个函数的指正的，程序运行过这段代码即回收掉这段内存。但是可以用把它复制给一个变量，让这个变量充当这个指针。

JS的对象特性：封装、继承-读书笔记

构造函数、实例、prototype

在介绍封装和集成之前，需要了解 JS 对象的三个概念： 构造函数、实例 和 prototype。

以生产某个型号A的机器人为例。生产机器人我们首先需要一个设计好的模具(构造函数，java 里叫做类)，才能够按照这个模具成产出一个一个机器人(实例)，你可以给其取名 a、b、c、d(对象变量名，实际是指针)。另外，你可以根据匹配磨具生产出唯一的一套配件外设(prototype,即原型)。当你买了一个机器人 a 的时候(创造了一个对象 a)，你可以说对象有模具中的功能，也有提配件供外设提供的功能。

在 JS 中，要实例化某个对象，先要声明一个构造函数(模版，类),实例化一个实例(对象)就是按照模版中的代码在新开辟一块内存，然后让你的变量名指向这块内存。prototype(原型)是所有实例和构造函数共享的一套方法和属性，有一块自己独立内存，构造函数的内存和实例的内存中都有一个指针指向 prototype的内存。

JS对象：函数-读书笔记

Function 概述

函数Function 是 JS 预设好的一种对象类型，函数名即指向这个对象的一个指针，所以可以把函数名当做其它函数的参数。

JS的函数是没有重载的。

函数对象默认定义好、主要的属性和方法有如下：

  • arguments 和 callee
  • this
  • call()、apply() 和 bind()
  • prototype
