关于单打的一些球路（持续更新…）

发接发

  1. 对对手要有一个了解，判断前后场能力，对手后场能力差果断发高远球。
  2. 观察对方接发站位，比较靠前可以增加偷后场/接发弹、推后场的次数。
  3. 抢高点放中路网前是非常有效的接发球方式，线路短，这个时候基本只需要防两条线，回搓和挑后场，更多的重心放在前场，要成功地挑到底线，球一定会很高，不然就是半场。
  4. 如果对方发外角，谨慎推直线或者就近放网，最好的办法是对角，包括勾对角和推对角底线过顶，尤其是推过顶，专治发外角。
  5. 如果对方发平高后场，准备动作请将肘部抬高，因为这样可以减少你上手球的准备时间。
  6. 对方发平高，有点被动的情况，最好直接顶一个很高的后场，给自己足够的准备时间，最好是中路，中路最安全。
  7. 发网小球后，注意力主要集中在网前和对手推两边线上。

网前球

  1. 对于网前比你差的人，放贴网球很有效，特别是带旋转的。
  2. 对于网前比你好的人，尽量少放网，要放选择中路球或者远网球。
  3. 对方吊你反手网前，如果上网充分，可以勾对角到你的正手网前，对于业余来说，正手总是比较容易打的。

多拍相持

  1. 多拍阶段要耐心，等待机会出现后，再果断加速突击。
  2. 位置不好，不要轻易吊球或者杀球，这样只能让自己被动。
  3. 中局多使用对角线，和重复落点是非常有效的套路，如果自己对于变线的感觉不好的时候，中路平高球就是最好的选择，等自己手感上来之后再去追求更大的角度。
  4. 连续大对角调动，特别是正手网前结合反手底线。结合一拍突如其来的正手底线，打乱对手的步伐。
  5. 开局一定要杀，要威慑对手，让对方脚步定死，然后才考虑劈杀两腰。

关于羽毛球的一些真·人生经验

技术篇

  1. 找个靠谱的教练，认真系统的学习一段时间，是业余真正的捷径；
  2. 多打单打，会比双打更容易提高个人能力；
  3. 打好基础，知道自己打出去的球会到什么位置，才能讲预判下一拍和战术；
  4. 刻意练习很重要，每一个技术动作都需要花时间、花心思、花精力去“练球”，才能掌握的好；
  5. 基础技术动作 —> 身体素质 —> 球路 —> 战术，升级道路大概是这样的；
  6. 良好的身体素质，是速度、力量、少失误的保障，所以需要你额外做很多无球的专项训练；

伤病篇

  1. 运动前动态拉伸（热身），运动后放松（静态拉伸），是减少意外伤病最好的办法；
  2. 业余80%的膝盖伤病，都来自于膝盖力量不足，靠墙静蹲是一个锻炼的好动作；
  3. 运动前，请认真评估自己的状态，疲劳、醉酒后打球，大概是拿生命在运动吧；
  4. 受伤后，请认真养伤，莫贪一时之乐；
  5. 适量、适度运动；

装备篇：

  1. 装备不一定要买最好的，“合适”才重要；
  2. 为了膝盖和脚，鞋子要买好的，尺码和宽楦要合适；
  3. 合适的毛巾袜和鞋垫占脚感体验很重要的一部分；
  4. 觉得球拍不好打？换根线、换个磅数试试；
  5. 压缩护腿对于大跑动量的人，确实有好处；
  6. 汗多手胶滑，试试护腕；

DNS 皮毛

这是一篇介绍 DNS（Domain Name System ） 皮毛的文章。

一、什么是 DNS

DNS = Domain Name System

我们平时访问网页，通常都是输入一个网站的域名（eg: https://google.com ），然后游览器会返回相应的网页给我们。

游览器拿到我们输入的域名，然后通过网络，最终找到相应的服务器拿到了我们要的网页，再返回给游览器。

在万维网中，服务器地址通常是用 IP 进行表示的。那么，我们的输入的域名，是怎么变成服务器的 IP 地址呢？

这个转换的过程，就是DNS 要做的。

学球回忆录4：单脚起跳杀球和吊球



杀球和吊球都是建立在良好的高球基础之上的。

杀球
  • 杀球：和高远球一样，小臂发力，摒住到腰下边。杀球后，是直线下去，再向左泄力。
  • 杀球问题：侧身太多，再加上倒拍，就侧身超过180度了，所以本身加倒拍应该不超过180度；找准击球点和位置之后，就不要跌小碎步了，不然容易错过最佳击球点;
  • 侧身后，重心在右脚上，右脚屈膝蓄能，左脚微微抬起。架拍，起跳转体，杀球。
  • 击球点要高，位置偏前。
  • 杀球要压得住球，注意落点和线路。
吊球
  • 正手吊直线和斜线。拍头要立起来，切多一点，接近90度，要跳起来。切的薄，切的速度快，球的速度就快，也贴网。
  • 击球使用偏拍头的位置。
练习方法

杀球：和高球一样多球喂球，球与球之间间隔不要太快，注重质量。
吊球：一个吊球，一人挑球。分位置进行多组。

flv.js框架解读（1）

知识储备

流媒体方面的知识

  • flv格式
  • mp4、fmp4格式

前端方面的知识

  • H5的video、audio标签
  • ES6
  • MSE

核心实现

  • video标签
  • MSE

FLV（Flash Video）是 Adobe 推出的一种视频格式，既可以用flash 播放，又可以用其他非flash 播放器播放。

video标签是 H5的一个新媒体标签，即视频播放标签。但是video标签不支持直接播放flv 的视频。

MSE（Media Source Extensions），即流媒体扩展 API。 提供了实现无插件且基于 Web 的流媒体的功能。使用 MSE，媒体串流能够通过 JavaScript 创建，并且能通过使用 audio 和 video 元素进行播放。

所以flv.js核心做的事情是：将获取到的flv 流媒体，经过格式转换，然后通过MSE 接口，喂给video标签。

学球回忆录3：高远球

高远球是最难的内容，也是练习的最多的部分。

基本从第二节课开始到结束，绝大部分时候都是高远球的练习。

要点
  1. 高远，45度侧身站位，转右脚，不需要太多引拍调整，直接翻拍小臂内旋发力，头前屏住，收起架拍。
  2. 主要爆发力，是依靠小臂的内旋，手腕需要摒住。
  3. 击完球的瞬间，只需要轻微向前自然抖一下，卸掉力就可以了，然后自然让拍子在身前架住。
  4. 脚上的动作主要是交叉步。重心在右脚上，屈膝引拍，击球的瞬间，右脚起跳、转体，击球。
  5. 为什么会丢掉击球点(高、前)：架好拍瞄准后，击球时不需要再提前增加侧身和多余的倒拍，那样只能让你本来已经判断好的击球点丢失掉。
  6. 侧身不需要过多，45度左右为佳。
练习方法

2组定点高远多球练习。不用加上脚上动作，定点击球，左右区各20个为一组，2组。

2组起跳高远球（起跳是为了抢高点和用上蹬跳转身的力量）。练习数量同上。

交叉起跳的练习。站在底线两根线上练习，一脚踩一根线，右脚在后。重心在右脚上，左脚抬起 —> 转身，交换脚的位置，左脚落地，右脚抬空 —> 右脚落地，交换脚位置，复原至起始位置，重复练习。

学球回忆录2：网前攻防

第二个内容就是网前攻防。

所谓网前攻防，就是一个人在网前连续下压封网，另外一人则是下手球的被动摆脱。

主要提高的是网前基本封网能力，以及下手被动球的摆脱能力。

技术要点
  1. 网前攻防。攻站在前发球线左右，先小步下压，再跟进下压。防，中场偏后，一拍多一个手柄长度离双打后第二个根发球线，以左脚为轴心防守。手指发力拿来顶，不需要翻腕。
  2. 扑球，要压上网前去；防，当反手被压制时，可以拐个对角。
  3. 关于防守。位置被攻调动开后，要快速回位，恢复两脚基本平行的站位。
  4. 关于进攻。要有预判意识，预判对方下一拍可能的位置，甚至可以提前移动去抓球。
练习方法

基本的练习：1:1，一个封网，一人防守。15min 一换。

进阶练习：2:1，1人封网，2人防守。10min 一换。

1对2要点。扑球人要有预判，了解对方两个人回球难易位置，对下一个球有预判。防守两脚平行站位，根据不同位置进行蹬跨和回位。防守方战术方面，两个人回直线2次左右调动对方单侧站位，然后快速拐对角进行被动球摆脱。
